Интерпретатор языка `.zxc`

Этот проект представляет собой простой интерпретатор пользовательского языка программирования, написанный на C++.
Язык поддерживает базовые типы данных, арифметику и вывод значений в консоль.

Возможности

Поддержка типов переменных:

  `int` — целые числа, поддерживаются арифметические операции `+ - * /` и скобки `()`.
  `string` — строки, задаются в кавычках `"..."`.
  `bool` — логические значения (`true` / `false` или `0` / `1`).

Поддержка арифметических выражений**:

  Приоритет операций: `*` и `/` выполняются раньше, чем `+` и `-`.
  Поддержка скобок `( ... )`.

Команда вывода:

  ```zxc
  print(x)
  ```

  Позволяет вывести в консоль значение переменной или строковый литерал.

Пример кода на `.zxc`

```zxc
int a = 5
int b = 10
int c = a * (b + 2)

string msg = "Результат: "
bool flag = true

print(msg)
print(c)
print(flag)
```

Вывод:

```
Результат: 
60
1
```

## Ограничения

* Нет поддержки `if/else` или циклов.
* В строках не поддерживаются экранированные символы.
* Нельзя смешивать типы данных в выражениях.
* Переменные должны иметь корректные имена (без спецсимволов и пробелов).
